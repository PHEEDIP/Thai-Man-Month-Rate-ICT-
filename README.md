# เครื่องคำนวณ Man-Month Rate — กลุ่มวิชาชีพ ICT

> เครื่องมือคำนวณค่าตอบแทนที่ปรึกษาด้านเทคโนโลยีสารสนเทศและการสื่อสาร (ICT)  
> อ้างอิงหลักเกณฑ์ราคากลางการจ้างที่ปรึกษา **ตารางที่ 4** ฉบับสำนักงานบริหารหนี้สาธารณะ (สบน.) เดือนมีนาคม ๒๕๖๙

DEMO <https://pheedip.github.io/Thai-Man-Month-Rate-ICT-/>
---

## วัตถุประสงค์

ใช้สำหรับ**หน่วยงานราชการ รัฐวิสาหกิจ และองค์กรของรัฐ**ในการประมาณการค่าใช้จ่ายบุคลากรที่ปรึกษา ICT เพื่อประกอบการจัดทำ **TOR (Terms of Reference)** และ **ราคากลาง** ในการจัดซื้อจัดจ้างตามระเบียบกระทรวงการคลัง

---

## ขอบเขตการใช้งาน

- คำนวณ **Billing Rate / Man-Month Rate** ของที่ปรึกษา ICT แต่ละตำแหน่ง
- รองรับประเภทที่ปรึกษาครบทุกประเภทตามหลักเกณฑ์ ได้แก่
  - บริษัทที่ปรึกษา (มีหลักฐาน ISO / ซอฟต์แวร์ / ประกันภัย ครบ 1–3 รายการ)
  - ที่ปรึกษาอิสระ (ผ่านองค์กรรัฐ หรือตรง/Freelance)
  - พนักงานรัฐ / สถาบันการศึกษา (บุคลากรหลักและบุคลากรผู้ช่วย)
  - บุคลากรสนับสนุน (ช่างเทคนิค / เลขานุการ)
- แสดง Mark-up Factor, Basic Salary, Man-Day, Man-Hour และค่าตอบแทนรวม
- **ส่งออกรายงาน** ในรูปแบบ Microsoft Excel (.xlsx) และ PDF (A4 แนวนอน)

---

## สูตรที่ใช้คำนวณ

```
Man-Month Rate  =  Basic Salary × Mark-up Factor

Mark-up Factor  =  (Basic + ค่าสวัสดิการสังคม + ค่าโสหุ้ย) × (1 + Professional Fee%) ÷ Basic
```

| ประเภทที่ปรึกษา | Mark-up Factor |
|---|---|
| บริษัท (ครบ 3 หลักฐาน) | 2.560 |
| บริษัท (2 หลักฐาน) | 2.464 |
| บริษัท (1 หลักฐาน) | 2.430 |
| บริษัท (ไม่มีหลักฐาน) | 2.074 |
| ที่ปรึกษาอิสระ (ผ่านองค์กรรัฐ) | 1.430 |
| ที่ปรึกษาอิสระ (ตรง / Freelance) | 1.000 |
| พนักงานรัฐ / สถานศึกษา (บุคลากรหลัก) | 1.750 |
| พนักงานรัฐ / สถานศึกษา (บุคลากรผู้ช่วย) | 1.467 |
| ช่างเทคนิค (อัตราคงที่) | 39,120 บาท/เดือน |
| เลขานุการ (อัตราคงที่) | 22,050 บาท/เดือน |

> 1 man-month = 22 วันทำการ = 176 ชั่วโมง

## การใช้งาน

1. เปิดไฟล์ `index.html` ด้วย Web Browser (Chrome / Edge / Firefox)
2. เลือก **ประเภทที่ปรึกษา** และ **วุฒิการศึกษา / ประสบการณ์** ของแต่ละตำแหน่ง
3. ระบุจำนวน **Man-Month** ที่ต้องการ
4. ระบบจะคำนวณ Billing Rate และยอดรวมโดยอัตโนมัติ
5. กดปุ่ม **"ส่งออก Excel"** หรือ **"ส่งออก PDF"** เพื่อบันทึกรายงาน

## หมายเหตุ

- เครื่องมือนี้จัดทำขึ้นเพื่อ**อำนวยความสะดวก**ในการประมาณการเบื้องต้นเท่านั้น
- ก่อนนำไปใช้ในเอกสารราชการ กรุณาตรวจสอบตัวเลขกับหลักเกณฑ์ฉบับที่มีผลบังคับใช้อย่างเป็นทางการทุกครั้ง
- อัตรา Basic Salary อ้างอิงจากตารางที่ 4 กลุ่มวิชาชีพ ICT ในหลักเกณฑ์ดังกล่าว
